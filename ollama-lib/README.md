# Ollama Lib

A Rust library for integrating with Ollama AI models.

## Overview

This library provides a clean, type-safe interface for interacting with Ollama's REST API, enabling developers to easily integrate AI capabilities into their Rust applications.

## Features

- Type-safe API client
- Async/await support
- Comprehensive error handling
- Extensible architecture

## Installation

Add this to your `Cargo.toml`:

```toml
[dependencies]
ollama-lib = "0.1.0"
```
